# Product Metrics & Funnel Analysis (SQL Project)

## Project Overview
This project focuses on analyzing user behavior data to understand product funnel performance and conversion metrics. The goal is to identify drop-offs, calculate conversion rates, and analyze user cohorts for product optimization.

## Dataset
The dataset contains event-level user activity including:
- User ID
- Event type (visit, signup, purchase)
- Event timestamp
- Platform
- Country

## Key Analysis Performed
- Funnel stage user count analysis
- Conversion rate calculation using SQL window functions
- Drop-off analysis
- Cohort grouping based on signup date

## Tools Used
- MySQL
- MySQL Workbench
- SQL (CTEs, Window Functions, Aggregations)

## Key Insights
- Identified major drop-offs between funnel stages
- Calculated step-wise conversion percentages
- Grouped users into cohorts to analyze signup trends

## Files Included
- funnel_analysis.sql (SQL queries)
- Screenshots folder (Query outputs and results)
